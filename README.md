# Guitar Practice

Aplicativo web para praticar riffs, solos e músicas no violão/guitarra. Isole trechos, ajuste a velocidade e automatize repetições.

## 🚀 Funcionalidades

- 🎸 Cadastro de músicas personalizadas com suporte a áudio local (MP3, WAV, etc.)
- ⏱️ Loop preciso com visualização de waveform
- ⚡ Velocidade de 50% a 150%
- 🔄 Repetições configuráveis ou loop infinito
- ⏳ Contagem regressiva antes do treino
- ⏸️ Pausa entre loops

## 🎹 Atalhos de Teclado

| Tecla | Ação |
|---|---|
| `Espaço` | Pausar / Iniciar |
| `Esc` | Cancelar rotina |
| `S` | Marcar início do loop |
| `E` | Marcar fim do loop |
